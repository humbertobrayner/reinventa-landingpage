# Reinventa+

O **Reinventa+** é uma iniciativa para ajudar profissionais a **reinventarem sua carreira** e explorarem novas oportunidades no setor de tecnologia. Nosso objetivo é oferecer **capacitação acessível e direcionada**, conectando pessoas às demandas do mercado digital e às melhores práticas do setor.

## Tecnologias utilizadas na landing page
- **Frontend:** HTML, CSS, TypeScript
- **Estilização:** TailwindCSS  

## Instalação
Clone o repositório e instale as dependências:

```bash
git clone (https://github.com/humbertobrayner/reinventa-landingpage/)
cd reinventa-mais
npm install
```

## Como usar
Rode o projeto localmente:

```bash
npm run dev
```
